Parse a valid XML file using XPath.

This project demonstrates JavaScript features such as:
  • Reading an XML file using FileReader
  • Reading an XML file using Fetch
  • Selecting an XML file using Drag and Drop
  • Parsing an XML file into a DOM Document using DOMParser
  • Searching for a string value, node attribute, node, list of nodes, or other values using XPath 1.0 and Document.evaluate
  • Displaying a value, node, or list of nodes that have been found using XPathResult
  • let, class, arrow functions, rest parameters, template strings, tagged template strings, and others.
This project demonstrates the CSS feature:
  • Numbering sections using CSS Counters
